A continuacion, hare una lista con todos los comandos que usaremos de Git en PowerShell:

Git init: crea un nuevo repositorio
Git clone https://github.com/########.gi: Clona un repositorio remoto
Git add nombre_archivo.txt: añade un documento o carpeta
Git add .: Añade todos los documentos
Git commit -m "mensaje del commit": Hacer un commit
Git status: Actualiza lo hecho 
Git push -u origin main: envia el proyecto al repositorio.
